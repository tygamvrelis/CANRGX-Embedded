Reads from the magnetometer and stores the results in the myMPU object passed in.

Note
The high and low bytes switch places for the magnetic field readings due to the way the registers are mapped. Note that 7 bytes are read because the magnetometer requires the ST2 register to be read in addition to other data
Parameters
myMPUPointer to the data structure which stores the data read from the MPU9250 sensor
semHandle for the semaphore to take while transferring data
Returns
1 if successful, otherwise a negative error code
