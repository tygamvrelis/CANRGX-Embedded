◆ taskFlags

uint8_t taskFlags = 0x00
static

Used to track which sensors have fresh data in the buffer.

When a bit is 1, it indicates fresh data has been received from its corresponding sensor. All the bits are cleared once a transmission is initiated. The bit-to-sensor mapping is as follows:

  • Bit 0: accelerometer data
  • Bit 1: magnetometer data
  • Bit 2: control signal data
  • Bit 3: temperature sensor data
