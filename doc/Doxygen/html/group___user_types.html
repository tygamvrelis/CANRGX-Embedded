◆ controllerStates_e

enum controllerStates_e

Enumerates the states the controller can be in. Each corresponds to a specific function with which the duty cycle for the magnets will be varied.

Enumerator
EXPERIMENT0 

Baseline run: no magnetic field, no TECs

EXPERIMENT1 

Same as EXPERIMENT0

EXPERIMENT2 

Magnet1: +1, Magnet2: +1, TECs on (DC anti-Helmholtz)

EXPERIMENT3 

Magnet1: +1, Magnet2: -1, TECs on (DC Helmholtz)

EXPERIMENT4 

Magnet1: +1, Magnet2: 0, TECs on

EXPERIMENT5 

AC trapezoid anti-Helmholtz (in-phase)

EXPERIMENT6 

AC trapezoid Helmholtz (180 degrees out of phase)

EXPERIMENT7 

Magnet1: sin, Magnet2: cos

EXPERIMENT8 

Same as EXPERIMENT2

EXPERIMENT9 

Same as EXPERIMENT3

EXPERIMENT10 

Same as EXPERIMENT0

IDLE 

No experiment; idle signals
