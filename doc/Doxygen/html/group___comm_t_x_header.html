void commTXSendPacket ( TickType_t *  lastWakeTime,
TickType_t *  cycleStartTick 
)

Initiates an asynchronous packet transmission after 3 ms have passed since the last time.

When this function is called, it first waits until 3 ms have passed since the last time a packet transmission was initiated. Then, it clears the task flags and gets the current OS tick. The OS tick is loaded into the packet and stored in the cycleStartTick variable, hence indicating that the next cycle has started. These steps ensure deterministic 3 ms timing. An 87 microsecond wait occurs (to (guarantee an interpacket delay of about 2 bytes), then finally the asynchronous transfer is initiated. The TX thread is then blocked until the callback occurs or 3 ms have elapsed (timeout).

Parameters
lastWakeTimePointer to the OS tick at which vTaskDelayUntil was last called
cycleStartTickPointer to the OS tick at which the current transmission cycle began
