void commRXEventHandler ( void  )

Upon receiving a full buffer of bytes, this function will be invoked from the PC RX thread to parse the data in the buffer and take the appropriate action depending on the message.

There are 4 cases:

  1. Manual override start: notify the control task and pass in the experiment start number with the manual override start bitmask
  2. Manual override stop: notify the control task with the manual override stop bitmask
  3. Manual hard reset: enter a critical section, set control signals to idle, wait 1 s for the magnetic field to drain its energy, then perform a full system reset
  4. Invalid case; ignore it
