◆ ADC_buff

volatile uint32_t ADC_buff[3][ADC_DATA_N]

The data buffer into which ADC samples are asynchronously transferred via DMA.

Since the ADC samples are 12 bits, each element of this array holds 2 samples. For example, ADC1 reads from channels 0 and 1 in sequence, so for n an even integer and m an odd integer, ADC_buff[0][n] would contain channel 0 data and ADC_buff[0][m] would contain channel 1 data
