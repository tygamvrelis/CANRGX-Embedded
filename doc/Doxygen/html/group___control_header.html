◆ controlEventHandler()

void controlEventHandler ( uint32_t  notification)

Given a task notification value, this function extracts the command or event information from it, then conditionally passes the extracted event to processReceivedEvent for handling.

The call to processReceivedEvent occurs if the received event is different than the current event, or the received event is manual override start

Parameters
notificationThe notification value passed to the control task
