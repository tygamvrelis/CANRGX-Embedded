uint32_t  period,
float  percentOn,
float  amplitude

Implements an AC trapezoid.

The waveform looks something like this over 1 period, centered about the value 0:

*           /--------\
*          /          \
*                      \          /
*                       \--------/
*
Parameters
timeThe input time at which the function is to be evaluated
periodThe period of the whole AC form in ms
percentOnThe % of the period for which the amplitude is constant Value must be in range [0.0, 100.0]
amplitudeThe amplitude of the first "peak" in [-1.0, 1.0] (the amplitude of the second "peak" will be the opposite sign)
Returns
The value of the function given valid parameters, otherwise NAN. The return value will always be in range [-1.0, 1.0] given valid parameters

float  amplitude 
)

Function with which a periodic sawtooth waveform can be evaluated.

Computes y = A(2(t + phi*T/360)/T - 1), where t = time, T = period phi = phase in degrees

Parameters
timeThe input time at which the function is to be evaluated
periodThe period of the sawtooth in ms
phaseThe waveform's positive phase offset, in degrees
amplitudeThe amplitude in [-1.0, 1.0]
Returns
The value of the function given valid parameters, otherwise NAN. The return value will always be in range [-1.0, 1.0] given valid parameters
