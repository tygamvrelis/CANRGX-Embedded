Detailed Description

Container for control signal data.

Note
Each member is equal to the value of the original duty cycle (between -1 and 1) multiplied by 10,000 and then truncated to an integer. This allows precision to 2 decimal places without using floats, as the precision of floats was not necessary for these particular quantities
