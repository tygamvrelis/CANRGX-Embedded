void MPU9250EventHandler ( MPU9250_t *  myMPU9250)

Based on the latest accelerometer readings, this function determines which state we are in: reduced gravity, or none. If the state is different than the last time this thread ran its cycle, an event is generated which notifies the control thread (to start/stop experiments).

Parameters
myMPU9250Pointer to the data structure which stores the data read from the MPU9250 sensor
